# 🛡️ Advanced Custom Captcha Library - Mr.Rm19

Selamat datang di repositori **Advanced Custom Captcha**. Proyek ini adalah kumpulan metode verifikasi manusia (Human Verification) yang interaktif, unik, dan ringan. Dirancang menggunakan teknologi murni **HTML5, CSS3, dan JavaScript**, sehingga mudah di-host di platform seperti GitHub Pages tanpa memerlukan backend database yang rumit.

##  Fitur Utama
Proyek ini terdiri dari 6 modul verifikasi unik yang memiliki mekanisme berbeda:

1.  **Puzzle Slide:** Menggeser potongan gambar ke posisi yang tepat pada kanvas.
2.  **Image Grid Selection:** Memilih gambar berdasarkan kategori objek yang diminta secara dinamis.
3.  **Rotate Challenge:** Memutar objek yang miring hingga mencapai posisi tegak lurus (0 derajat).
4.  **Pattern Lock:** Menghubungkan titik-titik grid untuk membentuk pola keamanan tertentu.
5.  **Drag & Drop:** Interaksi fisik menyeret objek ke wadah target yang ditentukan.
6.  **Logic Semantic:** Menguji pemahaman konteks antara teks instruksi dengan pilihan ikon visual.

##  Teknologi yang Digunakan
* **HTML5 & Canvas API:** Untuk manipulasi gambar dan penggambaran pola secara real-time.
* **Modern CSS:** Menggunakan CSS Variables dan Flexbox/Grid untuk tampilan yang responsif.
* **Vanilla JavaScript:** Logika verifikasi murni tanpa ketergantungan pada library pihak ketiga (No jQuery/React).
